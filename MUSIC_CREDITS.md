# Music Credits

This project does not include sample audio files by default.

## Recommended Free Music Sources

For testing, please download music from:

1. **Free Music Archive** - https://freemusicarchive.org/
   - License: Various Creative Commons licenses
   
2. **Bensound** - https://www.bensound.com/
   - License: Creative Commons Attribution (CC BY)
   
3. **ccMixter** - http://ccmixter.org/
   - License: Creative Commons licenses

## How to Add Music for Testing

1. Connect your Android device via USB
2. Enable file transfer (MTP mode)
3. Copy MP3/M4A/FLAC files to the device's Music folder
4. Launch the app and grant storage permission
5. Your songs will appear in the library

## Supported Formats

- MP3
- M4A / AAC
- FLAC
- WAV
- OGG
